# MediFInd🔍💊

AI Medicine Search is a lightweight web application that allows users to upload an image of a medicine strip, bottle, or packaging and receive information about the medicine using AI image analysis.

The application uses the **Gemini 2.0 model through the OpenRouter API** to analyze the uploaded image and generate relevant information about the medicine.

This project focuses on simplicity: the AI directly interprets the image and returns structured information about the detected medicine without requiring OCR or complex preprocessing.

---

## 🚀 Features

* 📷 **Image-Based Medicine Search**
  Upload an image of a medicine strip, tablet, or packaging to analyze it.

* 🤖 **AI-Powered Analysis**
  Uses **Gemini 2.0 via OpenRouter** to analyze the image and identify the medicine.

* 💊 **Medicine Information Output**

  * Medicine name
  * Common uses
  * Possible side effects
  * General warnings

* ⚡ **Lightweight & Fast**

  * Pure **HTML + CSS + JavaScript**
  * No backend required

* 🌐 **Fully Client-Side**
  Runs as a static web application.

---

## 🧠 How It Works

1. User uploads an image of a medicine.
2. The image is sent to the **OpenRouter API**.
3. The **Gemini 2.0 vision model** analyzes the image.
4. The model returns structured information about the medicine.
5. The results are displayed on the interface.

## ⚙️ Setup & Usage

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/ai-medicine-search.git
```

---

### 2️⃣ Open the project

Simply open:

```
index.html
```

in your browser.

No build step required.

---

### 3️⃣ Add your OpenRouter API key

Inside `script.js`, replace:

```javascript
const API_KEY = "YOUR_OPENROUTER_API_KEY";
```

with your OpenRouter API key.

You can get an API key from:

https://openrouter.ai/
